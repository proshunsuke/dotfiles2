# Dotfiles

This is a dotfiles repository.

## Installation

```sh
% git clone git@github.com:proshunsuke/dotfiles.git
% cd dotfiles && make
```

## Platform

* Linux
  * based on Debian

## Try

Let's try in docker container.

### based on Debian

```sh
% make up-debian
% make install-in-debian
% make ssh-debian
```
